Politique de confidentialité

    - Modern File Explorer n'a pas accès à votre connexion Internet
    - Modern File Explorer ne fonctionne pas demander des données personnelles des utilisateurs.
    - Nous ne collectons pas de données personnelles des utilisateurs.
    - Nous ne conservons pas les données personnelles des utilisateurs.
    - Les e-mails qui nous sont envoyés ne seront pas divulguées ou vendues à des tiers
    - Des idées ou des demandes de fonctionnalités partagées dans les emails envoyés à nous deviennent notre propriété
    - L'utilisation de Modern File Explorer ne nécessite aucune transmission de données personnelles.

Nous nous réservons le droit de modifier cette politique à une mise à jour mais nous n'avons aucune intention de le faire en ce moment.
